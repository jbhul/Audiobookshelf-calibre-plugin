<h2>About Audiobookshelf Sync</h2>
<p>A calibre plugin to synchronize metadata from Audiobookshelf to calibre.</p>
<p>The source code of this plugin can be found <a href="https://github.com/jbhul/Audiobookshelf-calibre-plugin">on GitHub</a>.</p>
<p>If you encounter any issues with the plugin or have any feedback/suggestions, please submit them <a href="https://github.com/jbhul/Audiobookshelf-calibre-plugin/issues">here</a>.</p>
